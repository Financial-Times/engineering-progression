Careers and progression for engineers in the CTO organisation at the FT.

Overview

This site and the accompanying repository is the single source of truth for the competencies and level definitions for Engineering roles in the CTO organisation at the FT.

You can find more information on the GitHub repository

Competencies

Engineering competencies are used to inform conversations about career progression between an engineer and their line manager:

  • Access competencies data via API
  • Track progress with your manager or report via Google Sheets
  • View competencies as YAML in the repository
